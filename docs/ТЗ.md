Листы выгрузки данных:

2.	Ежедневные фин отчеты. Выгружается ежедневно за предыдущий день вручную с ЛК селлера, раздел «Финансовые отчёты» - «Ежедневные». Необходимо настроить выгрузку по API ежедневно за предыдущий день с дозаписыванием.

Блок «ПРОГНОЗ ПО ВМЕНЕННЫМ ДАННЫМ»
Ввод вмененных данных для подсорта: 
BG3:BP3 – диапазон для ввода вмененных данных по увеличению коэффициента спроса на модель, применяется к средней скорости продаж.
BG4:BP4 – диапазон для ввода вмененных данных по % выкупа на модель, применяется в расчете потребности на период. Для того, чтобы вмененный % выкупа применился в расчете, необходимо проставить галочку в ячейке BH4.
BF – столбец для ввода вмененных данных по процентному соотношению цветов от черного внутри модели. Для того, чтобы вмененный % цвета от черного применился в расчете, необходимо проставить галочку в ячейке BF28.
BG – столбец с 30 строки рассчитывает прогнозную скорость заказов по формуле ЕСЛИ: если стоит галочка в BF28, то среднесуточная фактическая скорость только по черному цвету модели со столбца AC умножается на вмененный процент цвета от черного со столбца BF, если галочки нет, то формула возвращает среднесуточное фактическое по цвету с AC, и далее все это умножается на коэффициент модели с диапазона BG3:BP3.
Столбец BH со строки 30 заполняется вручную для ручного вменения среденесуточной скорости заказов.
Столбец BI со строки 30 рассчитывается как Вмененная среднесуточная скорость в расчет, по формуле, где черный цвет равен значению со столбца BH, а другие цвета по формуле: если есть галочка в ячейке BF28, то к значению со столбцов BH применяется коэффициент цвета от черного со столбца BF внутри модели.
Столбцы BJ:BP с 30 строки рассчитывает какой сток необходим по вмененным данным и вмененной размерной сетке по формуле: произведение по ячейкам прогноз скорости с ячейки BG, если нет галочки в BI28, или со столбца BI, если есть галочка в BI28, на % размера в цветомодели на листе «Процентная разбивка размеров» , умноженное на цикл возврата B7 плюс произведение по ячейкам прогноз скорости с ячейки BG на % размера в цветомодели на листе «Процентная разбивка размеров» , умноженное на произведение период минус цикл возврата B4-B7 умноженное на формулу если: если в BG28 стоит галочка, то умноженное на вмененный % выкупа с диапазона BJ, если нет галочки, то умноженное на фактический % выкупа со столбца D.
BQ:BW – столбцы с 30 строки рассчитывают сколько необходимо допоставить по размерам по вмененным данным. По формуле «Какой сток необходим по вмененным данным и вмененном % размерной сетки» BJ:BP  минус остатки. Если остатков больше, чем в BJ:BP на конец периода, то формула возвращает 0.
